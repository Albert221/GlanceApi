# Glance API

Simple API written in Go for the [Glance app](https://github.com/Albert221/Glance).

## Installation

Make sure to export `PORT`, `SECRET`, `DBHOST`, `DBUSER`, `DBPASSWORD` and `DBNAME`.

```bash
go build -o glance-api main.go
./glance-api
```
